Not one book. But lots of short and focused books for people who make websites. Titles include "Content Strategy for Mobile", "CSS3 for Web Designers" and "HTML5 for Web Designers"

Hardboiled Web Design

Hardboiled Web Design is different. It's for people who want to understand why, when and how to use the latest HTML5 and CSS3 technologies in their everyday work.

Eloquent JavaScript

A Modern Introduction to Programming. Eloquent JavaScript is a book providing an introduction to the JavaScript programming language and programming in general. Available as a free e-book or in print.
